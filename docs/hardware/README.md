# Анализ железа

Раздел по железу проекта «Деньгожуй» (XIAO ESP32-S3 Sense): распайка, выбор
сенсоров/моторов, разбор плат-кандидатов, методы ориентации и построения модели
пространства. Часть документов — обзорные исследования (датированы, выбор позже
уточнялся в практике), часть — канон, синхронизированный с прошивкой.

Шаблон оценки новой платы — [`TEMPLATE.md`](TEMPLATE.md).

## Распайка и пины

- [**Распайка XIAO + TB6612 + VL53L7CX (+ MPU6050)**](wiring-xiao-motor-tof.md) — **канон**,
  синхронизирован с `drive_config.h`: пины моторов, I²C-шина D9/D10 (ToF + IMU), LPn, питание,
  чеклист перед включением.
- [**Схема подключения MPU6050 (GY-521)**](wiring-xiao-mpu6050-gy521.svg) — нарисованная схема
  IMU курса на общей I²C-шине (`.pdf`/`.png`/`.svg`, генератор `build_mpu6050_wiring.py`).
- [**Разводка и пины: анализ дефицита**](pinout-wiring-2026.md) — карта всех 11 пэдов XIAO, что
  занято приводом/ToF/IMU, где упрёшься при расширении, пути A (расширители) / B (гибрид).
- `build_wiring.py` — генератор схемы привода (schemdraw).

## Сенсоры и ориентация

- [**ToF VL53L7CX — лучшие практики**](vl53l7cx-best-practices.md) — конфиг, интерпретация
  `target_status`, фильтры качества, проекция зон в 3D + аудит нашего `xiao_tof.h`.
- [**Ориентация и картирование — решения**](orientation-and-mapping-decisions.md) — курс (IMU
  MPU6050/BNO085, без магнитометра), occupancy grid, когда реально нужен LiDAR. Decision-oriented.
- [**Сенсоры для робота (2026)**](sensors-2026.md) — обзорный каталог датчиков по категориям с
  привязкой к этапам автономности. _Обзор июнь-2026; конкретный выбор позже уточнён практикой
  (VL53L7CX вместо VL53L0X, MPU6050 как курс) — см. два документа выше._
- [**Моторы с энкодерами**](motors-2026.md) — выбор мотор-редукторов с AB-энкодерами (TT Encoder
  1:90, 4680 PPR) под `drive_config.h`, сколько брать на 4WD, токи, второй TB6612.
