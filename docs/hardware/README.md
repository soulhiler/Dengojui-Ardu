# Анализ железа

Разбор плат-кандидатов для проекта «Деньгожуй» (XIAO ESP32-S3 Sense): что это,
подходит ли под текущие задачи (камера+MJPEG, PDM-мик, привод TB6612, BLE,
всё на Arduino), какие есть аналоги, выводы и рекомендации.

Шаблон новой оценки — [`TEMPLATE.md`](TEMPLATE.md).

## Рассмотренные платы

| Плата | Класс | Вердикт | Разбор |
|-------|-------|---------|--------|
| Wireless-Tag ESP32P4C61-TINY | ESP32-P4 + ESP32-C61 (краудфандинг) | Избыточно/преждевременно — остаться на S3 | [esp32p4c61-tiny.md](esp32p4c61-tiny.md) |

## Текущее железо проекта

XIAO ESP32-S3 Sense — single-chip Arduino-решение: камера OV2640 (DVP, `esp_camera`,
MJPEG), PDM-микрофон, Wi-Fi 4 + BLE 5.0, microSD, зарядка от 3.7 В. Закрывает все
текущие задачи робота. Подробности — в корневом `README.md`.
