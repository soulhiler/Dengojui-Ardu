# Анализ железа

Разбор плат-кандидатов для проекта «Деньгожуй» (XIAO ESP32-S3 Sense): что это,
подходит ли под текущие задачи (камера+MJPEG, PDM-мик, привод TB6612, BLE,
всё на Arduino), какие есть аналоги, выводы и рекомендации.

Шаблон новой оценки — [`TEMPLATE.md`](TEMPLATE.md).

## Обзорные исследования

- [**Платы для робота: MCU и SBC (2026)**](research-boards-2026.md) — сравнительный обзор
  двух классов плат (микроконтроллеры vs Linux-одноплатники), бенчмарки AI/зрения, ROS,
  камера/видео, питание, реальные OSS-проекты и рекомендации по гибридной связке MCU+SBC.
- [**Кандидат 1: бюджетно-мобильный Linux-компаньон**](candidate-1-budget-compute.md) —
  детальный разбор тира Pi Zero 2 W / Luckfox Pico Ultra W / Milk-V Duo: что реально умеют,
  как встраиваются в проект к ESP32, грабли, рекомендация первого шага к зрению.
- [**Сенсоры для робота (2026)**](sensors-2026.md) — обзор датчиков по категориям
  (расстояние/LiDAR, IMU/одометрия, контакт/линия, питание/среда/звук, 3D-глубина) с привязкой
  к этапам автономности: что включить на ESP32 сейчас, что добавить с Pi, что избыточно.
- [**Разводка и пины**](pinout-wiring-2026.md) — как сенсоры сходятся с реальными выводами
  XIAO ESP32-S3 поверх привода/камеры: карта пэдов, дефицит пинов (свободен только D10),
  конфликты I²C-адресов, варианты (I²C-расширители vs гибрид с Pi).
- [**Схема подключения (Путь B)**](wiring-path-b.md) — конкретная разводка гибрида XIAO + Pi +
  сенсоры: таблицы пинов, питание, I²C-адреса, линк ESP32↔Pi. Картинка:
  [diagrams/pinout-path-b.svg](diagrams/pinout-path-b.svg).
- [**Моторы с энкодерами**](motors-2026.md) — выбор мотор-редукторов с AB-энкодерами (чёрный TT
  Encoder 1:90, 4680 PPR), распиновка под `drive_config.h`, сколько брать на 4WD (2 энкодера),
  ток и второй TB6612.
- [**Питание и защита батареи**](power-and-battery-2026.md) — всё по питанию 6WD: 2S BMS (выбор,
  чего избегать), TVS/флайбэк/предохранитель, зарядка, токовый мониторинг (INA3221), согласование
  с драйвером DBH-12, схема, чек-лист и шоппинг-лист с ссылками Lazada.
- [**Голосовые модули**](voice-modules-2026.md) — голосовой стек для робота (захват→STT→понимание→TTS)
  и где что работает (ESP32/Pi/Jetson/облако): микрофонные массивы (reSpeaker/XVF3800), оффлайн-команды
  (ESP-SR, SEN0539), STT (Vosk/whisper), TTS (Piper), LLM-диалог; ⚠️ русский — только на Pi.

## Текущее железо проекта

XIAO ESP32-S3 Sense — single-chip Arduino-решение: камера OV2640 (DVP, `esp_camera`,
MJPEG), PDM-микрофон, Wi-Fi 4 + BLE 5.0, microSD, зарядка от 3.7 В. Закрывает все
текущие задачи робота. Подробности — в корневом `README.md`.
